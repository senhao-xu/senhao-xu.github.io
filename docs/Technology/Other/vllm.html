使用Vllm搭建DeepSeek ​

参考地址

前置条件

  • GPU
  • Docker GPU 驱动

下载模型 ​

两种下载方式，选择其一即可。

huggingface ​

Models - Hugging Face 查找要使用的模型

bash
# 安装 Git LFS
git lfs install
# 下载模型
git clone https://huggingface.co/deepseek-ai/DeepSeek-R1-Distill-Qwen-7B

MdoelScope ​

模型库首页 · 魔搭社区 提供多种下载方式

bash
# 安装 Git LFS
git lfs install
# 下载模型
git clone https://www.modelscope.cn/Tencent-Hunyuan/Hunyuan3D-2.1.git

启动 ​

yaml
  vllm:
    container_name: vllm
    restart: no
    image: vllm/vllm-openai:latest
    ipc: host
    volumes:
      - ./models:/models
    command: ["--model", "/models/DeepSeek-R1-Distill-Qwen-7B", "--served-model-name", "r1", "--max-model-len", "8912", "--max-num-seqs", "32", "--gpu-memory-utilization", "0.90"]
    ports:
      - 8000:8000
    deploy:
      resources:
        reservations:
          devices:
            - driver: nvidia
              count: all
              capabilities: [gpu]
